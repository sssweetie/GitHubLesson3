Последовательность выполнения Pull requests:
1. Выполняем Fork по ссылке удаленного репозитория
2. Клонируем репозиторий в локальную папку: git clone [ссылка] Название_папки
3. Открыаем эту созданную папку, записываем туда последовательность выполнения действий.
4. Сохраняем и коммитим изменения в этом файле.
5. Пушим изменения: git push
6. Переходим в удаленный репозиторий, выбираем вкладку Pull requests и выполняем это действие.
7. Делаем скриншот выполненного действия.
8. Загружаем домашку.
